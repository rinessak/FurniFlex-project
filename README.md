# FuniFlex

Full-stack web platform for managing company profiles, services, and export data. Built using PHP with MVC architecture and MySQL for data storage.

<h2>Project is based on mvc pattern</h2>

Explore platform online:

**[FurniFlex Platform](https://furniflex.kesug.com/)** 

## Tech Stack

- PHP with OOP (MVC structure)

- MySQL

- JavaScript

- Bootstrap, HTML, CSS, Twig

## Features

- Manage company profiles

- Store and update services and export information

- Structured backend using MVC pattern

- Database-driven content management
